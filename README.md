## 소개
<br/>
<br/>



로그인시 Spring Security를 통한 검증 구현

<br/>

Spring Security를 통해 보안측면을 다양하게 강화

<br/>



<br/>
<br/>
<br/>
<br/>




#### 주요 구현
<br/>

* 로그인 이후 권한 필터링을 통한 페이지 리다이렉션
* 특정 페이지 권한없는 계정 접근시 처리방법 구현
* csrf토큰 값과 bycrypt로 보안성 강화




<br/>
<br/>

#### 추가된 라이브러리
<br/>

* lombok
* MyBatis
* log4j2
